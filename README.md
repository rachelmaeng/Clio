# Clio

A mindful nutrition tracking app for iOS.

## About

Clio helps users reflect on nourishment, movement, and body state without shame or restriction. Built with SwiftUI and SwiftData.

## Features

- **Nutrition Tracking** - Log meals with awareness, not restriction
- **Movement Log** - Track Pilates, walks, strength, and rest days
- **Daily Check-in** - Capture how your body feels each day
- **Reflections** - Pattern insights based on your entries
- **Dual Modes** - Light and Evening atmospheres

## Principles

- No shame language
- No streaks or leaderboards
- Calories hidden by default
- Calm, reflective tone

## Requirements

- iOS 17.0+
- Xcode 15+
- Swift 6

## Build

```bash
open Clio.xcodeproj
# or
xcodebuild -project Clio.xcodeproj -scheme Clio -destination 'platform=iOS Simulator,name=iPhone 15'
```
